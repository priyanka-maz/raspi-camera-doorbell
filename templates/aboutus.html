About Raspi-Cam

Raspi-Cam is a door camera with face detection and recognition capabilities. It also has the ability to stream, record and capture still images from the live feed of the camera. This project was made on Raspberry Pi. Users may check the cctv feed of their house, from any network whether it be local network or outside the local network.

How it Works?


Raspi-Cam uses OpenCV and face_recognition for face detection, face recognition, video recording and image capturing. It uses Python Flask for backend. The frontend is made using HTML(jinja templates), CSS and Javascript. It uses ngrok to host the website and make it available for users to check the cctv feed remotely.

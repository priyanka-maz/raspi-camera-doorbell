About Raspi-Cam

Raspi-Cam is a door camera with face detection and recognition capabilities. It also has the ability to stream, record and capture still images from the live feed of the camera. The project is made on Raspberry Pi

How it Works?


Raspi-Cam uses openCV and face_recognition for face detection, face recognition, video recording and image capturing. It uses Python Flask for backend. The frontend is made using HTML, CSS and Javascript.
